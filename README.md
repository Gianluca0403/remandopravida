Remando Pra Vida — Landing Page + Loja

Landing page desenvolvida para o projeto social Remando Pra Vida, uma iniciativa que utiliza o skate como ferramenta de transformação social.
O site conta com apresentação do projeto, seções informativas e uma loja integrada, tudo feito usando HTML, CSS e JavaScript puro.

Landing Page:

- Seção de apresentação do projeto
- Explicação dos módulos: ensino de skate e skate nas escolas
- Galeria / Carrossel de fotos
- Área de contato com links diretos para WhatsApp e Instagram
  
 Loja:

- Exibição de produtos
- Design leve e responsivo
- Interação com botões e animações
- Espaço preparado para integração futura com checkout real

Responsividade

- O site é totalmente responsivo, adaptado para:
- Celulares
- Tablets
- Notebooks e desktops

FRONT END - TCC


No nosso projeto de TCC, eu fiquei responsável pelo front-end. Tentei fazer algo simples, mas que ao mesmo tempo fosse moderno e fácil de usar. Queria que o usuário tivesse uma experiência agradável sem complicação.

O sistema tem uma área de feedback, onde as pessoas podem enviar sugestões ou opiniões. Esses feedbacks são enviados pelo front-end para um banco de dados simples, onde ficam armazenados para a gente analisar depois.

No back-end, usei Node.js com o Express, que faz a ponte entre o front-end, o banco de dados e o servidor. É ele que recebe os dados que enviamos, como os feedbacks, e guarda tudo direitinho.

Além disso, pretendo integrar uma área de login com sessões para que os usuários possam se autenticar e acessar conteúdos exclusivos. Assim, o sistema fica mais seguro e personalizado.

Essa foi uma das minhas contribuições para o projeto.
